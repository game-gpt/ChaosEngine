## 什么是 Chaos Engine？

Chaos 引擎是一款专为 MMO 设计的新型架构。它摒弃了"开发者生产内容、玩家消费内容"的线性模式，转而构建由**玩家集体行为驱动**的动态世界。

**核心理念**：技术下沉、创意上浮。开发者从"世界建造者"转变为"规则立法者"，玩家的创造力成为内容生成的引擎。

---

## 核心特性速览

### 🌊 共识场 Consensus Field

将玩家行为转化为世界状态变迁的社会应力系统。

| 应力类型 | 描述 |
|:---|:---|
| S_combat | 战斗烈度 |
| S_trade | 贸易活跃度 |
| S_ecology | 生态平衡 |
| S_order | 秩序/混乱 |

当应力突破阈值 → 触发涌现事件（世界变化）

---

### 🎭 传奇系统 Legend System

告别职业与等级，玩家的成长由行为决定。

**六大维度**：
- **征伐之痕** - PVE/PVP 偏好
- **羁绊之痕** - 独行/团队
- **货殖之痕** - 生产/贸易
- **开拓之痕** - 安土/探索
- **天理之痕** - 秩序/混沌
- **地缘烙印** - 区域归属

玩家行为持续积累 → 突破阈值 → 涌现专属称号（如"商路开拓者"、"林地守护者"）

---

### ⚖️ 天理系统 Heavenly Principle

世界宏观参数，由玩家**共业**驱动。

| 参数 | 范围 | 驱动因素 |
|:---|:---|:---|
| 王朝气数 | [0, 1] | 官府任务、剿匪 |
| 灵气潮汐 | [-1, 1] | 术法技能使用 |
| 格物致知 | [-1, 1] | 创新技法、蓝图 |

运营团队仅需观察参数、投放催化剂，人力成本降至传统 MMO **5% 以下**

---

### 🎲 业力 Karma System

社会交互的应力单位，天理运算的基础。

- K_combat - 杀伐业力
- K_trade - 流通业力
- K_ecology - 生灭业力
- K_order - 秩序业力

---

### 🛡️ 气运系统 Fortune System

**双轨防御体系**，用经济学而非封禁对抗工作室。

**轨道一：气运剥夺**
| 等级 | 状态 | 效果 |
|:---|:---|:---|
| 1 | 时运不济 | 掉率/暴击率微降 |
| 2 | 财星隐匿 | NPC回收价降低 |
| 3 | 业力缠身 | 怪物仇恨异常 |
| 4 | 天厌人弃 | 制造品必带诅咒词条 |

**轨道二：诅咒举报**
玩家消耗传奇值主动举报 → 高可信度目标跳过缓冲期直接进入"业力缠身"

---

### ⚔️ 技法与连招系统

技能不是预设的，而是**发现**的。

- 约 30 种**动作原语**：挥砍、突刺、格挡、翻滚...
- **元素效应**：火焰、冰霜、雷电、毒素...
- 玩家通过特定动作序列触发**技法**
- 技法可记录、命名、交易 → 形成**知识资产 NFT**

---

### 🧬 动态生态系统

怪物是**种群**而非固定个体。

- **基因池**：每种怪物拥有基因频率分布
- **进化压力**：玩家行为驱动基因漂移
- **个体差异**：每只怪物独特
- **区域分化**：不同区域种群特征各异

---

### 🏭 制造与经济系统

**物品 = 材料属性向量 + 工艺参数 + 制作者印记**

- 无预设配方，通过**配方发现**和**蓝图 NFT**积累知识资产
- 产业链协作：原材料 → 半成品 → 成品
- **知识贬值曲线**：学习人数超阈值 → 制作成本下降，稀有性降低
- **归零事件**：王朝气数归零 → 资源重生，新玩家红利期

## 系统架构

```
┌─────────────────────────────────────────┐
│  宏观趋势层：世界趋势参数、历史演变      │
├─────────────────────────────────────────┤
│  资产记录层：知识资产 NFT、历史哈希     │
├─────────────────────────────────────────┤
│  涌现生成层：事件模板、上下文填充        │
├─────────────────────────────────────────┤
│  共识场层：应力计算、阈值监控           │
├─────────────────────────────────────────┤
│  基础交互层：移动、战斗、交易            │
└─────────────────────────────────────────┘
```

**数据流向**：
玩家操作 → 行为事件 → 共识场更新 → (阈值触发) → 涌现生成 → 资产记录 → 宏观趋势

|:---|:---|:---|:---|

| 内容填充 | 百人流水线 | **趋近于零** | 玩家共业驱动生成 |
| 反作弊 | 长期对抗 | **趋近于零** | 气运系统经济学瓦解 |
| 策划能力 | 填表工 | **通才型** | 设计因果、调控涌现 |

## 经济模型

- **价值基准**：玩家数量 = 世界价值
- **隐形税**：
  - 交易磨损：1%-3% 手续费（回收销毁）
  - 知识确权：铸造蓝图消耗活性材料

## 设计公理

1. **状态的不确定性是创造力的土壤**
2. **社会共识是虚拟世界真实性的锚点**
3. **知识资产是终极价值存储**
