# 战斗系统

## 概述

Chaos 引擎的战斗系统摒弃传统预设技能模式，构建基于**动作原语**与**元素效应**的组合发现空间。真正的强大技能来自玩家对世界底层规则的理解与创造性运用。

## 动作原语

游戏提供约 30 种基础动作原语，构成战斗的原子操作：

| 类别 | 动作原语 |
|------|----------|
| 武器动作 | 挥砍、突刺、上挑、下劈、横扫、格挡、弹反 |
| 身法动作 | 翻滚、滑步、跳跃、疾跑、下蹲 |
| 交互动作 | 蓄力、吟唱、瞄准、投掷 |

## 元素效应

由武器附魔、环境、药剂等赋予的临时属性：

```
火焰、冰霜、雷电、毒素、流血、眩晕、浮空、击退、标记、缠绕
```

## 技法发现机制

**技法**是玩家通过特定动作序列、时机和上下文条件触发的特殊效果。

### 发现流程

```
1. 尝试 → 玩家在游戏中尝试某种组合
2. 检测 → 系统实时分析输入序列和上下文
3. 验证 → 在实战中多次（3-5次）成功复现
4. 记录 → 系统提示命名，记录至资产层
5. 传播 → 可传授给其他玩家
```

### 技法示例

| 技法名称 | 动作序列 | 条件 | 效果 |
|----------|----------|------|------|
| 焰闪 | 翻滚 → (0.3s内) 突刺 | 武器附魔火焰 | 突刺附加火伤，路径留下火痕 |
| 雷返 | 格挡 → (被雷属性击中时) 突刺 | 成功格挡雷伤 | 释放范围雷暴 |
| 冰霜新星 | 跳跃落地 → (0.2s内) 横扫 | 武器附魔冰霜 | 范围冰伤并减速 |
| 浮空连斩 | 上挑 → 跳跃 → 下劈 | 力量属性达标 | 多段伤害，最后一击击倒 |

## 连招系统

**连招**是将多个技法以特定时序连接形成的攻击链。

- 连招成功执行带来更大收益（伤害加成、特殊效果）
- 系统检测有效连击序列后提示记录
- 连招可作为**个人知识资产**保密或公开分享

## 知识资产 NFT

### 资产化机制

```yaml
技法/连招蓝图 NFT:
  元数据:
    - 名称
    - 发现者署名
    - 发现时间
    - 动作序列哈希
    - 效果描述
  不包含:
    - 具体操作细节（需通过师徒传授或练习掌握）
```

### 经济价值

- **收藏交易**：NFT 本身可作为收藏品交易
- **版税机制**：发现者可设定传授费，其他玩家学习时支付版税
- **声望溢价**：发现者声望与技法实用性决定价值

## 反脚本设计

| 攻略 | 脚本 |
|------|------|
| 无法传授"手感"，需真实练习 | 固定连招会被怪物AI识破反制 |
| 知道≠能做到 | 不存在无脑站桩输出的最优循环 |
