# 制造系统

## 概述

Chaos 引擎彻底抛弃预设物品模型，物品由**材料属性、工艺参数和制作者印记**共同决定，形成连续高维的属性空间。

## 物品本质：属性向量

不存在预设的"物品"，每件装备由以下数据定义：

```json
{
  "item_type": "slashing_weapon",
  "materials": [
    {"id": "iron_ingot_east", "purity": 0.85, "hardness": 52, "toughness": 38},
    {"id": "leather_grip", "comfort": 25}
  ],
  "craft_parameters": {
    "quench_temp": 820,
    "hammer_blows": 115,
    "edge_angle": 22.5
  },
  "crafter_id": "0x7B2F...",
  "craft_timestamp": 1745059200
}
```

客户端根据属性向量实时计算最终数值并生成外观。

## 配方发现与蓝图 NFT

### 工艺参数空间

制造系统提供可调节的工艺参数：

| 工艺类型 | 参数示例 |
|----------|----------|
| 锻造 | 淬火温度、锤击次数、刃角 |
| 炼金 | 加热时间、搅拌速度、配比 |

参数与材料属性经非线性制造函数决定产出。

### 蓝图化流程

```
1. 探索 → 反复试验、记录结果、归纳规律
2. 验证 → 连续数次产出属性波动在可接受范围
3. 记录 → 生成蓝图（材料规格 + 工艺参数 + 预期产出）
4. 交易 → 蓝图可复制、交易
```

### 蓝图结构

```yaml
蓝图内容:
  材料规格: "硬度大于50的铁锭"
  工艺参数: 完整参数序列
  预期产出: 属性范围
  元数据:
    - 蓝图作者
    - 创建时间
```

## 产业链协作

| 层级 | 特点 | 适用场景 |
|------|------|----------|
| 个人手工 | 高随机性 | 探索新配方、制作原型 |
| 工坊生产 | 规模化、低变异 | 批量生产 |
| 产业链协作 | 专业分工 | 复杂装备（半成品组合） |

复杂装备催生玩家间分工：原材料生产 → 中间件加工 → 最终组装。

## 知识折旧曲线

### 价值驱动因素

```
最终产品价值 = 材料成本 + 蓝图知识价值 + 制作者声望溢价
```

### 自然通缩压力

- 玩家社群逐渐发现更高效配方
- 早期低级材料贬值
- 推动探索更高品质、更新颖属性市场

### 抗通胀设计

| 资产类型 | 作用 |
|----------|------|
| 金币 | 日常消耗、基础材料交易 |
| 知识资产 NFT | 核心价值存储 |
| 稳定币 | 大额交易（锚定稀有资源） |

工作室刷金对整体经济破坏力被大幅削弱。
