# 生态系统

## 概述

Chaos 引擎将怪物视为**种群**而非静态个体。每个区域的每种怪物拥有动态**基因池**，其遗传特征随时间、环境压力和玩家互动持续演化。

## 种群基因池模型

基因池由一系列**遗传特征**及其**频率分布**定义：

| 基因位点 | 等位基因 | 效果 | 初始频率 |
|----------|----------|------|----------|
| 体型 | 小型/普通/大型 | 生命值、攻击范围 | 80%/15%/5% |
| 攻击倾向 | 怯懦/谨慎/好斗/狂暴 | AI 行为树 | 10%/40%/40%/10% |
| 元素抗性 | 弱火/中性/耐火 | 火焰伤害承受 | 60%/30%/10% |
| 特殊能力 | 无/投网/呼唤同伴 | 额外技能 | 95%/3%/2% |

## 进化压力与基因漂移

基因频率改变由**选择压力**驱动：

### 压力来源

| 压力类型 | 触发条件 | 演化方向 |
|----------|----------|----------|
| 捕杀压力 | 玩家大量使用火焰法术 | "耐火"基因频率上升 |
| 食物压力 | S_ecology 应力过高（资源匮乏） | "大型"、"好斗"基因频率上升 |
| 社交压力 | "林地守护者"驱赶而非杀死怪物 | "谨慎"基因频率上升 |

### 演化周期

```
每24小时（现实时间）:
  1. 统计过去周期内狩猎数据
  2. 计算共识场应力
  3. 更新基因频率分布
```

## 个体怪物独特性

### 实例化过程

```
生成怪物时:
  1. 从基因池按频率分布随机抽取等位基因
  2. 组合成具体个体
  3. 施加极小概率随机突变
```

### 玩家体验

- **每只怪物独特**：无法通过名字和等级判断实力
- **必须观察**：通过战斗中的行为了解特性
- **区域差异**：不同区域种群因选择压力不同而分化

```
东部森林哥布林 → 普遍耐火
西部山脉哥布林 → 体型更大、更具攻击性
```

## 区域差异化

| 因素 | 影响 |
|------|------|
| 玩家狩猎偏好 | 当地种群针对性进化 |
| 环境资源 | 影响体型和攻击性 |
| 玩家角色类型 | "守护者"活动区域怪物更谨慎 |

## 生态知识资产化

### 研究活动

```yaml
解剖:
  操作: 击杀后解剖
  产出: 基因样本（标注来源区域、采集时间）

分析:
  条件: 拥有"怪物生态学"技能
  产出: 怪物生态分析报告
  内容: 当前区域基因频率概况
```

### NFT 化

- **报告 NFT**：详尽准确的分析报告可铸造出售
- **时效性**：报告准确性随种群演化而下降
- **持续需求**：创造"怪物生态学家"社会角色

## 反攻略与反脚本

| 传统攻略 | Chaos 生态 |
|----------|------------|
| "东部森林哥布林弱火" | 两周后可能"普遍耐火，建议改用冰霜" |
| 固定答案 | 调研方法 |

脚本依赖可预测的敌人行为，当每只怪物 AI 和抗性都不同且持续漂移时，固定逻辑脚本效率极低。
