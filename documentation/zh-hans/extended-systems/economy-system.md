# 经济系统

## 概述

Chaos 引擎构建以**知识资产**为核心价值存储的经济体系，通过经济学手段而非技术封堵实现反工作室防御。

## 价值锚定：玩家数量

传统 MMO 以资源产出锚定价值，Chaos 以**玩家共识**为价值锚点：

```
物品价值 = 社群共同相信的价值
事件历史 = 玩家社群共同见证、共同记忆
```

## 隐形税收模型

### 交易手续费

| 交易类型 | 手续费率 | 说明 |
|----------|----------|------|
| 普通交易 | 1% | 基础税率 |
| 高频交易 | 2-3% | 抑制套利脚本 |
| 知识资产交易 | 1.5% | NFT 交易税 |

### 税收用途

```yaml
税收流向:
  - 系统销毁（通缩）
  - 世界事件奖励池
  - 新玩家扶持基金
```

## 知识资产作为价值存储

### 资产类型

| 类型 | 形式 | 价值来源 |
|------|------|----------|
| 技法/连招 | 蓝图 NFT | 发现者署名、实用性 |
| 制造配方 | 蓝图 NFT | 稀缺性、产出品质 |
| 生态情报 | 报告 NFT | 时效性、准确性 |
| 历史事件 | 事件哈希 | 见证者证明 |

### 价值特征

```
知识资产价值 = 创造力 + 观察力 + 练习投入
```

- **不可复制**：NPH 难度，无法暴力穷举
- **可验证**：链上存证，可追溯
- **可交易**：市场流通，版税机制

## 反通胀设计

### 多层次货币体系

```
┌─────────────────────────────────────┐
│         知识资产 NFT（核心价值）        │
├─────────────────────────────────────┤
│         稳定币（大额交易锚定）          │
├─────────────────────────────────────┤
│         金币（日常消耗）               │
└─────────────────────────────────────┘
```

### 通缩机制

| 机制 | 效果 |
|------|------|
| 交易税销毁 | 减少货币流通量 |
| 知识资产沉淀 | 价值从货币转移到 NFT |
| 高端消费 | 金币消耗于非必需品 |

### 工作室免疫

```
工作室刷金 → 金币贬值 → 核心价值仍在 NFT → 刷金无利可图
```

## 声誉经济

### 信誉系统

```yaml
工匠声誉:
  来源: 产品质量、交付记录、客户评价
  效果:
    - 高声誉工匠产品溢价
    - 低声誉产品被市场抵制
    - "诅咒材料"被自发识别
```

### 信息不对称价值

- **鉴定师**：识别诅咒材料与干净材料
- **情报商**：提供区域生态、市场趋势信息
- **经纪人**：匹配买卖双方，降低交易成本

## 公地悲剧的自我修正

当脚本行为导致区域共识场恶化：

```
S_ecology 爆表 → 触发有害涌现事件（瘟疫、兽潮）
→ 所有区域玩家受损
→ 公会自发驱逐脚本
→ 保护共同"家园"
```

玩家利益与反作弊目标绑定，形成自发的社区治理。
