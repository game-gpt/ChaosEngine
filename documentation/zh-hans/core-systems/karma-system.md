# 业力系统

## 概念定义

业力是 Chaos 引擎中社会交互的应力单位，是天理系统运算的基础。将抽象的"社会应力"具象化为可感知的游戏概念。

**数学表达：F(x, y, z, t) = { K₁, K₂, K₃, ... }**

## 业力类型

| 业力类型 | 符号 | 描述 | 正向积累 | 负向积累 |
|:---|:---|:---|:---|:---|
| 杀伐业力 | K_combat | 战斗行为产生的应力 | 频繁战斗 | 和平活动 |
| 流通业力 | K_trade | 经济交易产生的应力 | 活跃贸易 | 交易停滞 |
| 生灭业力 | K_ecology | 生态交互产生的应力 | 破坏/消耗 | 滋养/恢复 |
| 秩序业力 | K_order | 社会行为产生的应力 | 建立秩序 | 混乱蔓延 |

## 业力计算规则

### 基础贡献值

| 行为类型 | 业力类型 | 基础值 | 修正因子 |
|:---|:---|:---|:---|
| 击杀怪物 | K_combat, K_ecology(+) | +0.0001 | 怪物等级、玩家数量 |
| 采集资源 | K_ecology(+) | +0.00005 | 资源稀有度、工具品质 |
| 种植/恢复 | K_ecology(-) | -0.0001 | 种植规模 |
| 玩家交易 | K_trade | +0.0002 | 交易金额 |
| 建造建筑 | K_order(+) | +0.005 | 建筑规模 |
| 破坏建筑 | K_order(-) | -0.01 | 建筑等级 |

### 空间分布

业力具有空间属性：

- 计算公式：**K(x, y, z, t)**
- 行为发生位置周围一定半径内生效
- 边界区域存在平滑过渡

## 业力作为应力单位

业力是共识场应力的具体表现形式：

| 概念层 | 抽象表达 | 具体表达 |
|:---|:---|:---|
| 数学定义 | 社会应力 S | 业力 K |
| 玩家感知 | 应力值 | 业力值 |
| 世界反馈 | 应力阈值触发 | 业力积累触发 |

## 业力与涌现

业力积累触发世界涌现事件：

1. **阈值检测**：区域业力值超过涌现阈值
2. **事件生成**：根据业力类型选择事件模板
3. **上下文填充**：结合贡献者图谱生成具体事件
4. **世界广播**：通知区域内所有玩家

## 共业驱动天理

全体玩家的业力总和驱动天理参数变化：

```
全局业力积分 → 天理参数更新 → 世界趋势演变
```

| 统计周期 | 计算方式 | 影响范围 |
|:---|:---|:---|
| 每日 | 聚合当日行为 | 局部参数微调 |
| 每周 | 积分汇总 | 天理参数更新 |
| 每月 | 趋势分析 | 宏观叙事走向 |
