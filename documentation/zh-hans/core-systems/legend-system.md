# 传奇系统

## 概念定义

传奇系统记录角色所有关键行为的多维向量，是角色在世界上留下的**因果指纹**。摒弃传统等级与职业的预设牢笼，实现"行为即历史，经历即力量"。

## 六大核心维度

| 维度 | 名称 | 描述 | 数据范围 |
|:---|:---|:---|:---|
| 征伐之痕 | 战斗倾向 | PVE 与 PVP 偏好，激进果敢或谋定后动 | [-1.0, 1.0] |
| 羁绊之痕 | 协作倾向 | 独行侠或团队轴心，引领者或协作者 | [-1.0, 1.0] |
| 货殖之痕 | 经济活动 | 生产者、贸易家或市场操纵者 | [-1.0, 1.0] |
| 开拓之痕 | 探索倾向 | 安土重迁或向往未知边疆 | [-1.0, 1.0] |
| 天理之痕 | 道德倾向 | 维护秩序或倾向混沌（非善恶，行为模式） | [-1.0, 1.0] |
| 地缘烙印 | 区域忠诚 | 对各区域势力的归属感与贡献度 | Map<区域ID, 权重> |

## 新手引导：职业倾向系统

### 设计目的

解决玩家初入沙盒的迷茫感，同时防止工作室脚本零成本套利。

### 核心机制

| 特性 | 说明 |
|:---|:---|
| 零数值加成 | 职业倾向不提供任何实际战斗属性或技能加成 |
| 零样本推荐 | 冷启动阶段利用倾向推送适合的基础引导任务 |
| 抗工作室设计 | 初始任务链包含非脚本化的人类交互 |

### 可选倾向

- 佣兵
- 游商
- 学者
- 游侠
- 方士

### 转化逻辑

当玩家真实行为开始覆盖初始倾向，传奇系统便会接管：

> 选择了"佣兵"但天天跑商的玩家，**货殖之痕**会盖过**征伐之痕**，系统顺势解锁"商路开拓者"称号。

## 传奇称号涌现机制

当传奇数据在某维度持续突破阈值并满足特定上下文，世界为玩家凝结**传奇称号**。

### 涌现示例

| 称号 | 触发条件 | 涌现效果 |
|:---|:---|:---|
| 绿林故交 | 森林活跃极久 + 多次清除脚本 | 动物不主动攻击，可感知业力失衡 |
| 角斗场之星 | PVP区域获得真人观众认可 | 解锁独特外观动作，自动发送观战邀请 |
| 商路开拓者 | 经济活动 > 0.7 + 连接三定居点 | 商队速度 +15%，可建临时贸易站 |

## 印记更新与衰减

- **实时更新**：相关行为发生时，对应维度微小移动
- **自然衰减**：长期不进行某类行为，印记向 0 回归
- **永久锚点**：重大事件（首杀世界Boss等）留下不可衰减的标记

## 角色叠加与冲突

- 一个玩家可同时拥有多个传奇称号
- 角色冲突时生成独特的**个人叙事任务**
- 解决冲突可融合出全新独一无二的称号
- 长期不活跃的角色进入**休眠状态**
