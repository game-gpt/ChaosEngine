# Chaos 引擎文档

欢迎来到 **Chaos 引擎** 官方文档。

Chaos 引擎是一款专为大型多人在线游戏设计的新型游戏架构。它摒弃了传统游戏开发中"开发者生产内容、玩家消耗内容"的线性模式，转而构建一个由玩家集体行为驱动的、持续涌现新内容与新叙事的动态世界。

## 核心特性

- **涌现式架构**：玩家行为驱动世界演化，而非预设内容消耗
- **低成本运营**：运营团队规模降至传统 MMO 的 5% 以下
- **反脚本免疫**：通过经济学手段而非技术封堵遏制作弊
- **知识资产化**：技法、蓝图等无形知识成为可交易的稀缺资产
